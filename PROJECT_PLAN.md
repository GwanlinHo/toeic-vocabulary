# 多益單字隨身卡 (TOEIC Flashcard Web) 實作與擴充計畫書

本專案旨在建立一個輕量、反應快速的靜態網頁，幫助使用者利用碎片時間複習多益高頻單字。透過 GitHub Pages 部署，並可完美整合至 iOS 捷徑中。

---

## 1. 專案目標與核心價值
- **極致輕量**：無需安裝 App，透過瀏覽器即可獲得原生 App 般的流暢體驗。
- **針對性練習**：根據多益證書等級（綠色、藍色、金色）篩選不同難度的單字庫。
- **沈浸式學習**：完整呈現單字拼寫、音標、詞性、中文解釋、片語、同反義詞及符合多益考題風格的例句。
- **多感官互動**：支援智慧型中英文交雜語音朗讀，輔助聽力與發音。

## 2. 單字庫擴充藍圖 (2026 Q2 階段性目標)

目前的單字量尚在起步階段，預計透過以下維度進行大規模擴充：

### 2.1 數量擴充目標
| 等級 | 目前數量 | 目標數量 (Q2) | 核心來源 |
| :--- | :--- | :--- | :--- |
| **🟢 綠色** | 250 | **800+** | NGSL (基礎常用詞), TSL 基礎篇 |
| **🔵 藍色** | 245 | **1,200+** | TSL 進階篇, 辦公室日常溝通詞彙 |
| **🟡 金色** | 1,250 | **2,500+** | 專業商務、法律、財經、高階管理術語 |

### 2.2 內容深度與情境標籤
除了單字本身，將為每個單字新增「情境標籤 (Context Tags)」，方便使用者按領域複習：
- **[Corporate]**：企業管理、人事、行政。
- **[Finance]**：銀行、投資、預算、會計。
- **[Logistics]**：採購、運輸、倉儲、供應鏈。
- **[Tech]**：資訊科技、電子通訊。
- **[General Business]**：會議、談判、市場開發。

## 3. 技術架構與自動化流程
- **前端繪製**：純 HTML5 + CSS3 (採用 Flexbox/Grid 佈局，確保手機端體驗)。
- **邏輯控制**：原生 JavaScript (Vanilla JS)，負責 JSON 資料讀取、隨機演算法及 DOM 更新。
- **資料生成 (AI-Driven)**：利用 LLM 智慧產出符合多益語境的高品質例句，並透過 Python 腳本 (`scripts/`) 進行自動化 JSON 封裝與清洗。
- **部署環境**：GitHub Pages (HTTPS 存取)。
