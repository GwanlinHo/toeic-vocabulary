# 多益單字隨身卡 (TOEIC Flashcard Web) 實作與擴充計畫書

本專案旨在建立一個輕量、反應快速的靜態網頁，幫助使用者利用碎片時間複習多益高頻單字。透過 GitHub Pages 部署，並可完美整合至 iOS 捷徑中。

---

## 1. 專案目標與核心價值
- **極致輕量**：無需安裝 App。
- **針對性練習**：綠、藍、金等級分類。
- **沈浸式學習**：音標、詞性、繁體中文解釋及商務例句。
- **多感官互動**：針對 iOS (Siri) 與 Android (Google) 優化自然發音。

## 3. 技術架構與自動化流程
- **資料生成 (AI-Driven)**：利用 LLM 智慧產出高品質商務例句。
- **自動化清理**：Python 腳本確保跨等級不重複。
