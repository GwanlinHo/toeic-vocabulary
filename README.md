# 多益單字隨身卡 (TOEIC Vocabulary Flashcards)

> 線上網址：[https://gwanlinho.github.io/toeic-vocabulary/](https://gwanlinho.github.io/toeic-vocabulary/)

這是一個輕量、反應快速的靜態網頁工具，旨在幫助使用者利用碎片時間複習多益 (TOEIC) 高頻單字。專為行動裝置優化，可完美配合 iOS 與 Android 桌面捷徑使用。

**注意：本專案之單字、解釋及例句均由 AI Agent 自動產生，可能存在錯漏或不夠完整之處，僅供輔助學習參考使用。**

## 各等級單字庫概覽

本專案將單字按多益證書等級分類，使用者可依自身目標進行針對性練習：

| 等級 | 數量 | 涵蓋領域 | 目標分數參考 |
| :--- | :--- | :--- | :--- |
| 綠色 | 800 字 | 基礎商務溝通、日常生活、旅行、餐廳點餐、社交互動常用語。 | 470 - 725 分 |
| 藍色 | 2,000 字 | 進階職場、IT/AI、專案管理、財務行銷、國際商務旅行。 | 730 - 855 分 |
| 金色 | 1,600 字 | 高階經營管理、法律契約、財務報表、精確語法表達。 | 860 分以上 |

> **註**：資料庫經過自動化去重處理，確保單字在不同等級間不重複出現，總計收錄 **4,400** 個獨立單字。

## 特色功能

- **智慧型語音優化**：針對 iOS (Samantha/Siri) 與 Android (Google) 進行語音優化，提供最自然的發音體驗。
- **台灣腔調語音鎖定**：自動優化台灣腔調 (zh-TW) 語音體驗，確保發音符合本地習慣。
- **已學會排除功能**：支援將熟悉的單字標記為「已學會」，系統將自動從複習清單中排除（資料儲存於本地瀏覽器）。
- **個人進度追蹤**：即時顯示各等級的學習進度（剩餘未學字數），讓學習更有成就感。
- **等級分類**：支援根據多益證書等級（綠色、藍色、金色）篩選不同難度的單字庫。
- **完整資訊**：提供單字拼寫、音標、詞性、中文解釋、片語、同反義詞及例句。
- **語音朗讀**：整合 Web Speech API，支援單字朗讀及「全卡片朗讀」，包含單字、詞性、中文解釋、片語及例句。
- **響應式設計**：針對手機端優化，介面簡潔、操作直覺。
- **隨機學習**：每次點擊「下一個單字」都會從選定等級中隨機抽取。

## 技術架構

- **前端**：HTML5, CSS3 (Flexbox/Grid), JavaScript (Vanilla JS)。
- **資料層**：靜態 JSON 檔案 (`data_*.json`)。
- **本地儲存**：利用 `localStorage` 儲存使用者標記的已學會單字。
- **部署**：支援 GitHub Pages 部署為靜態網站。

## 目錄結構

- `index.html`：網頁主體。
- `style.css`：視覺樣式與主題配色。
- `script.js`：資料讀取、隨機抽取及互動邏輯。
- `data_*.json`：按等級分類的單字資料庫。
- `scripts/`：用於產出與整理單字資料庫的 Python 腳本。

## 部署方式

1. 將本專案推送到您的 GitHub 儲存庫。
2. 進入 **Settings** > **Pages**。
3. 在 **Build and deployment** 下選擇 `main` 分支並儲存。
4. 稍等片刻即可透過 GitHub 提供的 URL 訪問網頁。

## 行動裝置捷徑設定建議

### iOS (iPhone/iPad)
1. 使用 Safari 開啟網頁。
2. 點擊下方的「分享」按鈕。
3. 選擇「加入主畫面」。
4. 或是將網頁加入 iOS 捷徑，設為「顯示網頁」，即可像 App 一樣快速複習。

### Android
1. 使用 Chrome 開啟網頁。
2. 點擊右上角的「三點」選單。
3. 選擇「加入主畫面」。
4. 網頁將以圖示形式出現在桌面，點擊即可快速存取。
